# Product Management System

## Sobre o Projeto

Este projeto é um sistema de gerenciamento de produtos desenvolvido utilizando HTML, CSS e JavaScript puro.

O sistema possui uma área de login para o administrador, um painel administrativo e uma página para o gerenciamento dos produtos cadastrados, armazenando todas as informações no LocalStorage do navegador.

---

## Funcionalidades

- Login do administrador.
- Verificação de autenticação.
- Painel administrativo.
- Página de perfil do administrador.
- Cadastro de produtos.
- Validação dos campos do formulário.
- Prevenção de produtos duplicados.
- Listagem dos produtos cadastrados.
- Exclusão de produtos.
- Logout do administrador.
- Persistência dos dados utilizando LocalStorage.

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage

## Como Executar

1. Faça o download ou clone o repositório:

```bash

```

2. Abra o projeto em sua IDE preferida.

3. Execute o arquivo `index.html` utilizando a extensão Live Server do VS Code ou diretamente pelo navegador.
